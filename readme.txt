Workaround to setup WooCommerce deliver to a different address as default.

== Description ==

This plugin provides a workaround to setup WooCommerce deliver to a different address as default.

It can work even when the WooCommerce hook is broken using the option to force using Javascript.

== Installation ==

1. Upload the plugin zip
2. Activate the plugin through the 'Plugins' screen in WordPress
4. Go to WooCommerce > Shipping manager to setup
